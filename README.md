# sudoku #

## Usage ##

### Pre-requisites ###

1. Install sass (sudo gem install sass)
2. Install compass (sudo gem install compass)

### Installation and start ###

1. Clone this repo
2. Run "npm start" in the root of your clone
3. Start to play sudoku
